LyrOn is a modern, lightweight browser extension designed to enhance your music listening experience. It seamlessly integrates with your active tabs (like YouTube) to display synchronized lyrics in real-time through a beautiful, non-intrusive Glassmorphism interface.

## ✨ Features

* **Real-Time Lyrics:** Automatically detects the currently playing song and fetches its lyrics.
* **Modern UI:** Built with a sleek Glassmorphism design that blends naturally with any website.
* **Light/Dark Mode:** Fully customizable viewing experience with a built-in theme toggle.
* **Seamless Integration:** Works quietly in the background on supported media platforms.
* **Manifest V3:** Built using the latest, most secure browser extension standards.

## 🛠️ Tech Stack

This extension is built with a modern frontend architecture to ensure high performance and maintainability:

* **Framework:** React 18
* **Language:** TypeScript
* **Styling:** Tailwind CSS v3
* **Bundler:** Vite
* **Extension Tooling:** CRXJS Vite Plugin
